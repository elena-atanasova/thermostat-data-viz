# Data Analysis and Visualisation of Household Heating Behaviour

This project analyses how a single home is heated over an eight-month period and how indoor temperature and humidity respond to that heating behaviour. It uses 10-minute interval data from temperature, humidity, and boiler-status sensors to examine patterns of heating activity and resulting indoor conditions. 

## Research Focus
The analysis investigates the relationship between household heating activity and indoor environmental stability. It focuses on how boiler usage varies across seasons and whether these changes correspond to measurable differences in indoor temperature and humidity.

All analysis and visualisations are in `analysis.ipynb`. 

## Data
Provided as part of a university data visualisation module.

## Tools
Python, pandas, matplotlib
